📖 自炊本管理ソフト (ZIP/CBZビューア)

概要

PythonのGUIライブラリであるttkbootstrapを使用して開発された、ローカルフォルダ内のZIP/CBZ形式の自炊本ファイルを管理・閲覧するためのシンプルなデスクトップアプリケーションです。

モダンなUIを提供し、ページめくりアニメーション、読書進捗の自動保存、および各種ソート機能を備えています。

主な機能

📂 フォルダベース管理: 指定したフォルダ内の全ての書籍ファイルを一覧表示します。

💾 読書再開機能: ファイルごとに読了ページを自動で記録し、次回起動時に続きから読み始めるか確認します（settings.jsonに保存）。

➡️ ページナビゲーション:

マウスホイール、キーボードの矢印キー（左右）、画面の左右クリックに対応。

設定でページめくりの方向（L→R/R→L）を変更可能。

✨ アニメーション: ページめくり時にスムーズなスライドアニメーションをオプションで適用可能。

⚙️ ファイルリストソート: ファイル名（拡張子除く）、更新日、ファイルサイズでのソートに対応。

🖼️ 対応画像形式: JPG, PNG, WEBP などの主要な画像形式をZIP/CBZ内から読み込み可能。

動作環境

OS: Windows, macOS, Linux (Tkinterが動作する環境)

言語: Python 3.x

🚀 インストールと実行方法

1. 必要なライブラリのインストール

本プロジェクトを実行するには、以下のライブラリが必要です。仮想環境の使用を推奨します。

# 必要なライブラリを一括インストール
pip install ttkbootstrap Pillow


2. ファイルのダウンロード

GitHubリポジトリをクローンするか、book_manager.py、README.md、LICENSE、.gitignoreの各ファイルをダウンロードして、一つのフォルダに配置します。

3. アプリケーションの起動

インストールが完了したら、以下のコマンドでアプリケーションを起動できます。

python book_manager.py


起動後、左側のパネルにある**「📁 フォルダを選択/履歴」**ボタンから、書籍ファイル（ZIP/CBZ）が格納されているフォルダを選択して利用を開始してください。
